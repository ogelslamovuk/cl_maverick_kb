
# Запуск киоска

## Суть

Запуск киоска включает установку `SellerAgent`, настройку переменных окружения, запуск bat-файлов, создание ярлыка для браузера и проверку работы весов, если они используются.

## Когда применяется

При первичном запуске или настройке киоска.

## Где находится

Файлы `SellerAgent` должны лежать в:

```text
C:\Maverick\Maverick\SellerAgent\
```

Проверочные страницы:

- `https://kiosk.mooon.by/start`
- `https://kiosk.mooon.by/products`

## Порядок действий

1. Скачать архив `SellerAgent.zip`.
2. Разархивировать архив по пути `C:\Maverick\Maverick`.
3. Проверить, что все файлы `SellerAgent` лежат в `C:\Maverick\Maverick\SellerAgent\`.
4. Скачать актуальные jar-файлы по ссылкам из файлов:
   - `seller-agent.link`
   - `seller-agent-updater.link`
5. Создать переменные окружения:
   - `%MAVERICK_HOME% = C:\Maverick\Maverick\SellerAgent`
   - `SELLER_AGENT_MODE = KIOSK`
6. Запустить bat-файлы:
   - `setup-sevice.bat`
   - `setup-seller-updater.bat`
7. Создать ярлык `open_browser.bat` на рабочем столе.
8. Сменить иконку ярлыка на `Maverick.ico`.
9. Запустить созданный ярлык.
10. Выбрать кассу.
11. Выбрать баннер стартовой страницы.
12. Если на киоске есть весы, перейти на `https://kiosk.mooon.by/products`.
13. Выбрать весовой товар.
14. Положить что-нибудь на весы.
15. Проверить, что значение веса на экране киоска изменилось.

Если значение веса не изменилось:

1. Освободить COM-порт 20: остановить нативные службы весов или отключить и включить весы.
2. Перезапустить службу `Seller Agent Spring Boot Service`.
3. Открыть браузер и перейти на `https://kiosk.mooon.by/start`.
4. Повторить проверку весов.

## Обязательные правила

- Все файлы `SellerAgent` должны лежать в `C:\Maverick\Maverick\SellerAgent\`.
- Для режима киоска переменная `SELLER_AGENT_MODE` должна быть равна `KIOSK`.
- После настройки нужно проверить выбор кассы, выбор баннера стартовой страницы и работу весов, если они есть.

## Риски и контроль

- Тема повышенного риска: техническая настройка.
- Неверный путь установки или переменные окружения могут помешать запуску киоска.
- Неработающие весы могут привести к ошибкам при продаже весовых товаров.
- Действия с COM-портом 20 и службами требуют осторожности и проверки на нужном рабочем месте.

## Частые ошибки

- Файлы `SellerAgent` лежат не в той папке.
- Не скачаны актуальные jar-файлы.
- Не создана переменная `%MAVERICK_HOME%`.
- `SELLER_AGENT_MODE` не установлен в `KIOSK`.
- Не перезапущена служба `Seller Agent Spring Boot Service` после проблемы с весами.
