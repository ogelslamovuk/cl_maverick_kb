
# Сортировка афиши

## Суть

Афиша на `mooon.by` сортирует события по разделам и фильтрам. Основные параметры сортировки: количество сеансов, дата выхода в прокат, дата события, пометка `Премьера`, тип события, возрастное ограничение, категория и тег.

## Когда применяется

При проверке порядка карточек в ленте событий, разделах `Сегодня`, `Завтра`, `На неделю`, `Скоро` и календаре.

## Где находится

- Стартовая страница: лента событий.
- Раздел афиши: `https://mooon.by/afisha/minsk`.

## Порядок действий

1. Определить, где проверяется сортировка: стартовая страница, раздел афиши, раздел `Скоро` или календарь.
2. Определить выбранный фильтр: `Все`, `Премьеры`, `mooon+`, `Детям`, `Эксклюзив` или `Эксклюзивно`.
3. Проверить параметры события, по которым оно должно попадать под фильтр.
4. Проверить порядок карточек по правилам для выбранного места и фильтра.
5. Для календаря отдельно проверить дату отображения события и наличие кнопки `Купить билет`.

## Обязательные правила

### Стартовая страница: лента событий

- `Все`: показываются все события с сеансами. Лента содержит 25 событий. Карточки идут слева направо, начиная с событий с наибольшим количеством сеансов. Если количество сеансов одинаковое, первыми идут более новые события по дате выхода в прокат.
- `Премьеры`: показываются события с пометкой `Премьера`, фильмы первой недели проката. Прокатная неделя считается с четверга по среду. Карточки идут слева направо, начиная с событий с наибольшим количеством сеансов.
- `mooon+`: фильтр по параметру `Тип` из карточки события в софте: `Альтернативный` и `Спецпроект`. Карточки идут слева направо по возрастанию даты события.
- `Детям`: фильтр по параметрам `Тип = Общий` и возрастному ограничению `0+`, `6+`, `12+`. Карточки идут слева направо, начиная с событий с наибольшим количеством сеансов. Если количество сеансов одинаковое, первыми идут более новые события по дате выхода в прокат.
- `Эксклюзив`: фильтр по категории `Эксклюзив`/`Эксперт` и тегу `Эксклюзив`/`Эксперт`. Карточки идут слева направо по возрастанию даты события.

### Раздел афиши: Сегодня, Завтра, На неделю

- `Все`: показываются все события, которые сегодня в прокате. Карточки идут слева направо и сверху вниз, начиная с событий с наибольшим количеством сеансов. Если количество сеансов одинаковое, первыми идут более новые события по дате выхода в прокат.
- `Премьеры`: показываются события с пометкой `Премьера`, фильмы первой недели проката. Прокатная неделя считается с четверга по среду. Карточки идут слева направо и сверху вниз, начиная с событий с наибольшим количеством сеансов.
- `mooon+`: фильтр по `Тип = Альтернативный` и `Тип = Спецпроект`. Карточки идут слева направо по возрастанию даты события.
- `Детям`: фильтр по `Тип = Общий` и возрастному ограничению `0+`, `6+`, `12+`. Карточки идут слева направо и сверху вниз, начиная с событий с наибольшим количеством сеансов. Если количество сеансов одинаковое, первыми идут более новые события по дате выхода в прокат.
- `Эксклюзив`: фильтр по категории `Эксклюзив`/`Эксперт` и тегу `Эксклюзив`/`Эксперт`. Карточки идут слева направо по возрастанию даты события.

### Раздел Скоро

- `Все`: показываются все события выбранного раздела. Карточки идут сверху вниз по дате выхода в прокат по возрастанию даты.
- `Премьеры`: показываются события с пометкой `Премьера`, у которых день начала проката ещё не наступил. Карточки идут сверху вниз по дате выхода в прокат по возрастанию даты.
- `mooon+`: фильтр по `Тип = Альтернативный` и `Тип = Спецпроект`. Карточки идут сверху вниз по дате выхода в прокат по возрастанию даты.
- `Детям`: фильтр по `Тип = Общий` и возрастному ограничению `0+`, `6+`, `12+`. Карточки идут сверху вниз по дате выхода в прокат по возрастанию даты.
- `Эксклюзивно`: фильтр по категории `Эксклюзив`/`Эксперт` и тегу `Эксклюзив`/`Эксперт`. Реализовано через тег `Exclusive`. Карточки идут сверху вниз по дате выхода в прокат по возрастанию даты.

### Календарь

- Все события отображаются в календаре в день старта проката.
- События отображаются в Минске и в Гродно.
- Если в дату старта проката есть сеанс в выбранном городе, событие отображается с кнопкой `Купить билет`.
- Если в дату старта проката нет сеанса в выбранном городе, событие отображается без кнопки `Купить билет`.
- Для сеансов после даты старта проката событие отображается в день сеанса с кнопкой `Купить билет` в том городе, где стоит сеанс.
- Эти условия также относятся к вкладкам `Сегодня`, `Завтра`, `На неделю`.
- Во вкладке `Скоро` отображаются все события с лейблом `Премьера`, независимо от выбранного города и наличия опубликованных сеансов.
- Если во вкладке `Скоро` у события есть опубликованные сеансы на любую дату, отображается кнопка `Купить билет`. Если опубликованных сеансов нет, кнопка не отображается.

## Риски и контроль

- Некорректные параметры события могут привести к неправильному попаданию карточки в фильтр.
- Ошибка в количестве сеансов, дате события или дате выхода в прокат меняет порядок карточек.
- Для `Эксклюзивно` в разделе `Скоро` нужно отдельно проверить реализацию через тег `Exclusive`.

## Частые ошибки

- Проверяют только фильтр, но не порядок карточек внутри фильтра.
- Путают дату события и дату выхода в прокат.
- Не учитывают правило первой прокатной недели для `Премьер`.
- Не проверяют наличие кнопки `Купить билет` отдельно для календаря и вкладки `Скоро`.
- Путают фильтры `Эксклюзив` и `Эксклюзивно`.
