# Сертификаты в Manager

Эта страница — вход в инструкции по сертификатам, которые проверяются и настраиваются в Manager / back-office.

## Что относится к Manager

В Manager проверяются:

- категории сертификатов;
- выпуск сертификатов;
- текущий остаток;
- операции, продажи, списания и отмены;
- списки товаров и комбо, разрешённых для сертификата;
- промокоды, если обращение оказалось не про сертификат.

- В разделе **Manager / back-office** она нужна потому, что проверка и настройка выполняются именно в Manager.
