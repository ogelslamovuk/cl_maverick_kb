
# Сертификаты в Manager

Эта страница — вход в инструкции по сертификатам, которые проверяются и настраиваются в Manager / back-office.

## Что относится к Manager

В Manager проверяются и настраиваются:

- категории сертификатов;
- карточки сертификатов и подарочных карт;
- выпуск номеров;
- сертификаты без номинала для печати пластика;
- правила применения;
- списки категорий билетов, товаров и комбо;
- прайсы и доступность продажи;
- абонементы и количественные сертификаты;
- отчёты по предоплаченным сертификатам.

## Основной алгоритм проверки

Если сертификат не работает или не сходится остаток, открывай основную инструкцию:

[Проверка и разбор проблем с сертификатами в Manager](../Сертификаты/Проверка%20и%20разбор%20проблем%20с%20сертификатами.md)

## Где находится

Путь:

```text
Manager → Сертификаты
```

Основные разделы:

| Раздел | Для чего используется |
| --- | --- |
| **Категории сертификатов** | Карточки сертификатов, правила, выпуск, списки и прайсы. |
| **Сертификаты без номинала** | Заготовки номеров для печати пластика до продажи. |
| **Отчёты** | Проверка предоплаченных сертификатов и связанных данных. |

## Что нельзя менять без регламента

- правила применения сертификата;
- списки разрешённых товаров, комбо или категорий билетов;
- признаки активности;
- прайсы;
- параметры НДС;
- данные выпуска и активации;
- бухгалтерские признаки, связанные с эмитентом.
