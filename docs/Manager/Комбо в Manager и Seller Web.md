
# Комбо в Manager и Seller Web

Комбо — это готовый набор товаров, который продаётся на кассе Seller Web или в киоске. В Manager настраивается карточка комбо, категория отображения, состав товаров и прайс.

## Где отображается комбо в Seller Web

В Seller Web есть отдельная вкладка **Комбо**. В ней отображаются:

- избранные товары;
- категории комбо;
- комбо-наборы внутри выбранной категории;
- корзина заказа справа.

![combo seller tab](../assets/media/manager/manager-combo-seller-tab-pub.jpg)

Если у комбо нет изображения, на кассе может отображаться техническая заглушка. Для товаров изображения управляются через карточку товара на Portal: картинку загружают на сервер или указывают ссылку. Отображение изображений именно для кассира остаётся отдельной зоной проверки.

## Где находится настройка комбо

Открой **Manager → Продукты → Справочные комбо**.

![combo card main](../assets/media/manager/manager-combo-card-main-pub.jpg)

Справочник содержит все заведённые комбо: действующие и недействующие.

## Основные параметры карточки

В карточке комбо заполняются:

- наименование;
- сокращённое наименование;
- категория;
- тип;
- признак включения;
- ставка НДС;
- аннотация;
- порядок;
- изображения;
- цвет.

Категория определяет, в какой папке комбо будет отображаться на кассе. Значения категории берутся из справочника категорий комбо.

## Активность и доступность продажи

Признак **Вкл** управляет активностью комбо. Но одного включения недостаточно: чтобы комбо продавалось, у него также должен быть настроен прайс.

!!! warning "Комбо связано с деньгами и НДС"
    Перед публикацией комбо проверь ставку НДС, цену и доступность в нужном канале продаж. Ошибка в прайсе или ставке влияет на продажу и учёт.

## Состав комбо

Во вкладке **Продукты** добавляются товары, которые входят в состав комбо.

![combo card products tab](../assets/media/manager/manager-combo-card-products-tab-pub.jpg)

Проверяй:

- что в состав добавлены правильные товары;
- что количество каждого товара соответствует набору;
- что состав не перепутан с обычной карточкой товара.

## Категории комбо

Категории комбо управляют папками отображения на кассе.

![combo category card](../assets/media/manager/manager-combo-category-card-pub.jpg)

Если комбо не отображается в ожидаемой папке Seller Web, проверь категорию в карточке комбо.

## Прайс комбо

Прайс задаёт цену комбо.

![combo price table](../assets/media/manager/manager-combo-price-table-pub.jpg)

Проверяй:

1. Комбо включено.
2. Комбо привязано к правильной категории.
3. Во вкладке **Продукты** задан состав.
4. В прайсе есть актуальная цена.
5. Ставка НДС выбрана корректно.
6. После настройки комбо отображается в Seller Web или киоске.

## Что не считать ошибкой

- Комбо может быть заведено в справочнике, но не продаваться, если нет прайса или оно выключено.
- В справочнике могут оставаться старые недействующие комбо.
- Аннотация может использоваться как внутренняя пометка в карточке и не обязательно отображается клиенту.

## Частые ошибки

- Включили комбо, но не добавили прайс.
- Задали состав товаров, но выбрали не ту категорию отображения.
- Ищут картинку комбо на кассе, хотя отображение изображений для кассира требует отдельной проверки.
- Меняют ставку НДС без проверки налогового правила.
