# Вспомогательные справочники в Manager

Вспомогательные справочники Manager хранят короткие списки значений, которые потом выбираются в других карточках и таблицах.

## Где находится

Открой **Общее → Справочники → Вспомогательные**.

![helper directories tabs](../assets/media/manager/manager-helper-directories-tabs-pub.jpg)

## Что хранится во вспомогательных справочниках

В этом разделе находятся простые списки значений, например:

- категории организаций;
- виды деятельности организаций;
- категории должностей;
- ставки НДС;
- города;
- другие категории и типы, которые выбираются в связанных карточках.

## Как это используется

Значения из вспомогательных справочников подтягиваются в выпадающие списки других разделов Manager.

Примеры связей:

- категория и вид деятельности выбираются в карточке организации;
- категория должности выбирается при заведении сотрудника или пользователя;
- ставка НДС используется в прайсах и товарных настройках.

## Порядок действий

1. Открой нужный вспомогательный справочник.
2. Проверь, нет ли уже нужного значения.
3. Если значения нет, добавь его через кнопку **+**.
4. Укажи название значения.
5. Сохрани запись.
6. Проверь, что значение появилось в выпадающем списке связанной карточки.

## Важно

!!! warning "Не переименовывать без проверки связей"
    Вспомогательный справочник выглядит как простой список, но его значения используются в других карточках, прайсах, отчётах и настройках. Перед изменением проверь, где значение уже выбрано.

## Частые ошибки

- Добавляют дубль вместо проверки существующего значения.
- Меняют название значения, не проверив связанные карточки.
- Ищут настройку в основной карточке, хотя значение заводится во вспомогательном справочнике.
