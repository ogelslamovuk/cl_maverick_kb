
# Заполнение прайса с НДС

## Суть

Для мероприятий, билеты на которые облагаются НДС, прайс заполняется вручную. В поле `Ставка НДС` нужно выбирать `20 %`; значение `Расч.` для таких мероприятий не используется.

## Когда применяется

При создании или редактировании прайса для мероприятий, по которым билеты должны продаваться с НДС: например, спортивные трансляции, футбольные матчи и другие специальные события.

## Порядок действий

1. Открыть прайс мероприятия, для которого билеты должны продаваться с НДС.
2. Для каждого типа билета указать итоговую цену для покупателя в поле `Цена с НДС`.
3. В поле `Ставка НДС` выбрать `20 %`.
4. Рассчитать `Сумма НДС` по формуле: `Цена с НДС × 20 / 120`.
5. Округлить сумму НДС до 2 знаков после запятой.
6. Рассчитать `Цена без НДС`: `Цена с НДС - Сумма НДС`.
7. Округлить цену без НДС до 2 знаков после запятой.
8. Проверить, что `Цена без НДС + Сумма НДС = Цена с НДС`.
9. Проверить все типы мест, которые есть в прайсе: `Single`, `Love Seat`, `Sofa`, `Wheelchair Space`, `Family Sofa` и другие.
10. Сохранить прайс после проверки всех строк.

Пример для билета 25.00:

| Показатель | Значение |
| --- | --- |
| Цена с НДС | 25.00 |
| Ставка НДС | 20 % |
| Сумма НДС | 25.00 × 20 / 120 = 4.1666 -> 4.17 |
| Цена без НДС | 25.00 - 4.17 = 20.83 |

## Обязательные правила

- Для мероприятий с НДС в поле `Ставка НДС` выбирать `20 %`.
- Значение `Расч.` для таких мероприятий не использовать.
- `Сумма НДС = Цена с НДС × 20 / 120`.
- `Цена без НДС = Цена с НДС - Сумма НДС`.
- Сумма НДС и цена без НДС округляются до 2 знаков после запятой.
- При округлении третья цифра после запятой от 0 до 4 оставляет вторую цифру без изменений.
- При округлении третья цифра после запятой от 5 до 9 увеличивает вторую цифру на 1.

## Риски и контроль

- Тема повышенного риска: НДС, деньги, учёт.
- Онлайн-оплата может не показать ошибку, но касса/Titan POS выделяет НДС.
- Перед сохранением проверить ставку `20 %`, сумму НДС, цену без НДС и все типы мест.
- Ошибка в ставке или сумме НДС может привести к неправильному учёту.

## Частые ошибки

- Выбирают `Расч.` вместо `20 %`.
- Заполняют только часть типов мест.
- Не проверяют равенство `Цена без НДС + Сумма НДС = Цена с НДС`.
- Не округляют сумму НДС и цену без НДС до 2 знаков.
