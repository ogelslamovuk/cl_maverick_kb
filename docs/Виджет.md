# Виджет

Виджет — это элемент интерфейса, позволяющий встроить функциональность продажи билетов на сторонние сайты, мобильные приложения и в социальные сети без перехода пользователя на основной сайт.

## Краткое описание

**Виджет CinemaLab (go2.by)** — это веб-компонент, размещающий продающий билеты портал на сторонних ресурсах: сайтах партнёров, мобильных приложениях, агрегаторах афиш. Пользователь проходит полный путь от выбора фильма до оплаты, не покидая сайт партнёра.

### Технология

Работает на основе **iframe** (Inline Frame) — HTML-компонент, который встраивает документы, видео и интерактивные медиафайлы внутрь страницы. По сути это «сайт в сайте» или «картинка в картинке».
