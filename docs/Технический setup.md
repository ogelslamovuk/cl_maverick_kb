# Техническая настройка

Раздел для технических сценариев, которые нужны перед работой продукта или оборудования.

## Что здесь есть сейчас

- [Запуск киоска](Киоск/Запуск%20киоска.md)

## Когда использовать

Используй этот раздел, если задача звучит как:

- «не запускается»;
- «нужно подготовить устройство»;
- «нужно проверить техническую настройку»;
- «киоск/терминал не готов к работе».
