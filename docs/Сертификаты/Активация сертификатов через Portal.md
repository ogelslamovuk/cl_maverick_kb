
# Активация сертификатов через Portal

## Суть

Сертификаты активируются через форму на портале `mooon.by`. Для сертификатов Maverick в поле `Компании` обязательно выбирается ООО "Кино Маяк", чтобы эмитент корректно попал в учёт.

## Когда применяется

Когда нужно активировать сертификаты из подготовленного Excel-файла.

## Где находится

Портал `mooon.by`: `Администрация -> Активация сертификатов`.

По умолчанию используется вкладка `Из файла`.

## Порядок действий

1. Подготовить Excel-файл в формате `.xlsx`.
2. Проверить, что файл соответствует шаблону и содержит две колонки: `UN сертификата` и `Номер сертификата`.
3. Проверить, что номер сертификата состоит из 16 цифр.
4. Зайти на портал `mooon.by`.
5. Открыть `Администрация -> Активация сертификатов`.
6. На вкладке `Из файла` загрузить подготовленный Excel-файл.
7. В поле `Компании` выбрать ООО "Кино Маяк".
8. В поле `Комментарий` при необходимости указать, для кого и по чьей просьбе активируются сертификаты.
9. Проверить дату активации.
10. Нажать `Сохранить`.

## Обязательные правила

- Файл должен быть в формате шаблона: `UN сертификата` и `Номер сертификата`.
- Номер сертификата должен быть 16-значным.
- Номинал или категория сертификата определяется системой по данным сертификата.
- В одном файле могут быть сертификаты разных номиналов.
- Для сертификатов Maverick в поле `Компании` обязательно выбирать ООО "Кино Маяк".
- Не использовать ручную активацию через Битрикс.
- Не активировать сертификаты в обход формы портала.

## Риски и контроль

- Тема high-risk: сертификаты и учёт.
- Перед сохранением проверить, что файл загружен, компания выбрана ООО "Кино Маяк", дата активации указана корректно.
- Некорректный выбор компании может привести к неправильному указанию эмитента в учёте.
- Обход формы портала может нарушить корректный учёт сертификатов.

## Частые ошибки

- Загружают файл не по шаблону.
- Не выбирают ООО "Кино Маяк" в поле `Компании`.
- Не проверяют дату активации перед сохранением.
- Пытаются активировать сертификаты через Битрикс или другим обходным способом.
