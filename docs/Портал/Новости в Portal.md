# Новости в Portal

Раздел `Новости` показывает материалы для сайта mooon.by и даёт действия для перехода, изменения и деактивации.

## Где находится

Portal → `управление` → `Новости`.

## Что есть на странице

На странице доступны:

- заголовок `НОВОСТИ`;
- фильтры `Активные` и `Неактивные`;
- кнопка `Добавить`;
- список материалов;
- тип материала, например `Новость`, `Промо`, `Статья`;
- название материала;
- действия `Перейти к`, `Изменить`, `Деактивировать`.

Действие `Перейти к` ведёт на публичную страницу материала на `mooon.by`.

## Как работать со списком

1. Открой `Новости`.
2. Выбери `Активные` или `Неактивные`.
3. Найди материал по названию.
4. Для проверки опубликованной страницы используй `Перейти к`.
5. Для правки используй `Изменить`.
6. Для снятия с публикации используй `Деактивировать`.

## Важно

!!! warning "Публикация влияет на сайт"
    Не изменяй и не деактивируй материал без подтверждения владельца контента. Действия на этой странице могут повлиять на публичный сайт mooon.by.

## Частые ошибки

- Проверяют только активные материалы и не смотрят вкладку `Неактивные`.
- Нажимают `Деактивировать` вместо `Перейти к`.
- Считают тип `Статья`, `Новость` или `Промо` взаимозаменяемым, хотя на странице они отображаются как разные типы материалов.
