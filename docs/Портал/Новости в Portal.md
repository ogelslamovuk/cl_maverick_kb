# Управление новостями через Portal

Раздел `Новости` используется для просмотра материалов mooon.by и перехода к действиям с новостями, промо и статьями.

## Где находится

Portal → `управление` → `Новости`.

![Список новостей в Portal](../assets/media/portal/portal_news_ru.png)

## Что есть на странице

- переключатели `Активные` и `Неактивные`;
- кнопка `Добавить`;
- карточки материалов;
- типы `Новость`, `Промо`, `Статья`;
- действия `Перейти к`, `Изменить`, `Деактивировать`.

`Перейти к` открывает публичную страницу материала на mooon.by.

## Проверка материала

1. Выбери `Активные` или `Неактивные`.
2. Найди материал по названию и типу.
3. Используй `Перейти к`, чтобы проверить опубликованную страницу.
4. Перед `Изменить` или `Деактивировать` убедись, что действие согласовано с владельцем контента.

## Форма добавления

`Добавить` открывает отдельную форму. `Назад` возвращает к списку без публикации.

![Поля добавления новости в Portal](../assets/media/portal/news_add_ru.png)

| Поле или кнопка | Назначение |
|---|---|
| `Заголовок` | Название материала в Portal и на связанном экране сайта. |
| `Описание` | Краткое описание материала. |
| `Тип` | Выбор типа материала; в форме по умолчанию показана `Новость`. |
| `Дата` | Дата материала. |
| `Ссылка` | Адрес перехода для материала. |
| `Баннер` | Путь к основному изображению или загрузка файла. |
| `Разрешение планшета` | Отдельное изображение для планшетного представления. |
| `Задний фон` | Путь или загрузка фонового изображения. |
| `Содержание` | Основной текст материала. |
| `Мета заголовок` | SEO-заголовок страницы. |
| `Мета описание` | SEO-описание страницы. |
| `Мета ключевые слова` | Ключевые слова страницы. |
| `Описание картинки` | Текстовое описание изображения. |
| `Загрузить` | Выбирает файл для соответствующего изображения. |
| `Добавить` | Создаёт материал; действие может сделать контент доступным на публичном сайте. |

На карточке активного материала `Изменить` открывает редактирование, `Деактивировать` переводит материал в неактивные, а `Перейти к` открывает его публичный адрес.

## Важно

!!! warning "Изменения видны на публичном сайте"
    Изменение или деактивация материала может повлиять на mooon.by. Не выполняй действие без согласования.

## Частые ошибки

- Проверяют только активные материалы.
- Нажимают `Деактивировать` вместо `Перейти к`.
- Не учитывают тип материала при поиске.
