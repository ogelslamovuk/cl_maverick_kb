
# Поиск билета в Portal

Экран `Поиск билетов` помогает найти продажу по данным чека или клиента.

## Где находится

Portal → `поиск` → `Поиск билетов`.

![Поиск билетов в Portal](../assets/media/portal/portal_ticket_search_ru.png)

## Режим и поля

Перед формой есть переключатели `Продажа` и `Заведение`.

Поля поиска:

- `Дата начала`;
- `Номер чека`;
- `Email клиента`;
- `Дополнительный чек`;
- `Телефон клиента`;
- `Дополнительный платежный номер`;
- `Фамилия`.

## Порядок поиска

1. Выбери `Продажа` или `Заведение` в соответствии с рабочим сценарием.
2. Заполни известные поля.
3. Нажми `Поиск`.
4. Для нового запроса используй `Сбросить`.

## Важно

!!! warning "Форма использует персональные и платёжные идентификаторы"
    Используй только те данные, которые нужны для рабочей задачи. Не переноси email, телефон и платёжные номера в открытые заметки или переписку.

Точный смысл переключателей и состав результата поиска требуют подтверждения. Не делай вывод о статусе продажи только по одной найденной строке.
