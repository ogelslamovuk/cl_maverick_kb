# Поиск, возвраты и очистка чеков в Portal

Portal содержит отдельные экраны для поиска билетов, сертификатов, промокодов, возврата билетов и очистки чека.

## Поиск билетов

Где находится: Portal → `поиск` → `Поиск билетов`.

![Поиск билетов в Portal](../assets/media/portal/portal_ticket_search_ru.png)

На странице есть переключатели:

- `Продажа`;
- `Заведение`.

Поля поиска:

- `Дата начала`;
- `Номер чека`;
- `Email клиента`;
- `Дополнительный чек`;
- `Телефон клиента`;
- `Дополнительный платежный номер`;
- `Фамилия`.

Доступные действия:

- `Поиск`;
- `Сбросить`.

## Поиск сертификатов

Где находится: Portal → `поиск` → `Поиск сертификатов`.

На странице есть переключатели:

- `UN`;
- `Email`.

Поле ввода: `Введите номер сертификата`.

На странице показана подсказка: `Введите 16-ти значный номер сертификата и нажмите “Поиск”`.

## Поиск промокода

Где находится: Portal → `поиск` → `Поиск промокода`.

Поле ввода: `Введите номер промокода`.

Доступные действия:

- `Поиск`;
- `Сбросить`.

## Возврат билетов

Где находится: Portal → `управление` → `Возврат билетов`.

На странице есть поле `Номер чека` и кнопки:

- `Поиск`;
- `Сбросить`.

Эта страница только открывает поиск по номеру чека. Правила возврата смотри в отдельной инструкции.

## Очистить чек

Где находится: верхнее меню `очистить чек` или карточка `Очистить чек`.

На странице показан список кассовых зон и кнопка `Очистить чек` рядом с каждой зоной. В текущем интерфейсе видны зоны:

- `Arena 1`;
- `Arena 5`;
- `Arena 6`;
- `Café 1`;
- `DANA 5`;
- `MCM-SUV-1`;
- `MCM-TCK-1`;
- `Palazzo 2`;
- `Palazzo 4`;
- `Palazzo 5`;
- `Palazzo 6`;
- `Triniti 2`;
- `Triniti 4`;
- `Triniti offsite 2`.

## Важно

!!! warning "Возвраты и очистка чека"
    Возвраты, поиск по персональным данным и очистка чека относятся к операциям повышенного риска. Не нажимай `Очистить чек` и не выполняй возврат без подтверждённого регламента.

## Частые ошибки

- Ищут билет только по номеру чека, хотя на странице есть дополнительные поля поиска.
- Путают поиск сертификата по `UN` и по `Email`.
- Нажимают служебные кнопки до проверки кассовой зоны или номера чека.
