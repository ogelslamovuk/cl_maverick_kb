# Каталог в Portal

Раздел `Каталог` содержит справочники продуктов, групп, НДС, единиц измерения, прайсов, цехов и списков продуктов.

<div class="kb-meta" markdown>
<div markdown>
<strong>Для кого</strong>
Сотрудник, который проверяет или поддерживает товарные справочники и настройки продаж.
</div>
<div markdown>
<strong>Когда применяется</strong>
Когда нужно открыть справочник продукта, цены, НДС, цеха или списка продуктов в Portal.
</div>
<div markdown>
<strong>Что получится</strong>
Открыта нужная таблица каталога с поиском, колонками и действиями.
</div>
</div>

## Где находится

Portal → `каталог`.

![Каталог в Portal](../assets/media/portal/portal_catalog_ru.png)

## Разделы каталога

На главной странице каталога доступны:

| Раздел | Что открывает |
|---|---|
| `Продукты` | таблица продуктов |
| `Кассовые группы` | справочник кассовых групп |
| `Группы продуктов` | верхний уровень группировки продуктов |
| `Категории продуктов` | категории внутри продуктовой структуры |
| `Типы продуктов` | типы продуктов |
| `Классы продуктов` | классы продуктов |
| `Свойства` | свойства продукта |
| `Ставки НДС` | справочник ставок НДС |
| `Единицы измерения` | единицы измерения |
| `Прайсы продуктов` | прайсы и периоды действия |
| `Цеха` | цеха для ресторанных/производственных сценариев |
| `Списки продуктов` | списки продуктов |

## Общие инструменты таблиц

В таблицах каталога видны общие действия:

- поиск по таблице;
- `Очистить поиск`;
- `Показать/Скрыть колонки`;
- `Переключить полноэкранный режим`;
- сортировка колонок;
- действия с колонкой;
- выбор количества строк на странице;
- переход по страницам;
- кнопка добавления сущности.

## Продукты

В таблице `ПРОДУКТЫ` видны колонки:

- `№`;
- `Объект`;
- `Имя`;
- `Активный`;
- `Видимый`;
- `Публичный`;
- `Сокр. наименование`;
- `Порядок`;
- `Штрихкод`;
- `Единица измерения`;
- `Тип продуктов`;
- `Класс продуктов`;
- `Категория продуктов`;
- `Группа продуктов`;
- `Кассовая группы`;
- `Свойство`;
- `Ставка ндс`;
- `Цвет`;
- `Весовой`;
- `Вес тары`;
- `Маркированный`;
- `Маркированный SI`;
- `Маркированный UKZ`;
- `Возрастное ограничение`.

## Прайсы продуктов

В таблице `ПРАЙСЫ ПРОДУКТОВ` видны колонки:

- `№`;
- `Имя`;
- `Активный`;
- `Начало действия`;
- `Окончание действия`.

## Ставки НДС

В таблице `СТАВКА НДС` видны колонки:

- `№`;
- `Имя`;
- `Значение`.

## Цеха

В таблице `ЦЕХА` видны колонки:

- `№`;
- `Имя`.

В текущем списке видны значения `KITCHEN`, `BAR`, `MAIN`.

## Важно

!!! warning "Каталог влияет на продажи и учёт"
    Продукты, прайсы, НДС, цеха и списки продуктов могут влиять на кассу, киоск, Waiter, отчёты и налоговые настройки. Не создавай и не меняй значения без подтверждённого регламента.

## Частые ошибки

- Меняют продукт, не проверив прайс и период действия.
- Ищут настройку цеха в Waiter, хотя справочник цехов доступен в каталоге Portal.
- Скрывают колонку и считают, что данных нет.
- Используют справочник ставок НДС как налоговый регламент, хотя он показывает только видимые значения в системе.
