# Аналитика в Portal

Раздел `Аналитика` содержит три дашборда с разными задачами и наборами показателей.

| Дашборд | Для чего используется | Подробное описание |
|---|---|---|
| `Кинопульс` | Выручка, билеты, кинобар, ресторан, кафе, услуги и магазин | [Кинопульс в Portal](Кинопульс%20в%20Portal.md) |
| `Ресторан` | Выручка, гости, чеки, средний чек и разбивка по дням недели | [Ресторанная аналитика в Portal](Ресторанная%20аналитика%20в%20Portal.md) |
| `Маркетинг` | Программа лояльности, активность базы, регистрации и экономика баллов | [Маркетинговая аналитика в Portal](Маркетинговая%20аналитика%20в%20Portal.md) |

## Общие элементы

- Меню `аналитика` открывает список `Кинопульс`, `Ресторан`, `Маркетинг`.
- Быстрые периоды меняют временной интервал всего дашборда.
- Значок календаря открывает выбор произвольного интервала.
- Подпись с датами показывает фактические границы выбранного периода. Проверяй её перед чтением цифр.
- Зелёная стрелка показывает рост относительно сравнительного периода, красная — снижение.
- Подпись `С периода ...` указывает тип периода, с которым сравнивается карточка.
- Переключатели `Метрики`, `Таблица`, `График` меняют представление внутри конкретного блока, а не всего Portal.

## Как читать дашборд

1. Выбери нужный дашборд.
2. Задай период и проверь отображаемые даты.
3. Прочитай название блока и единицу измерения каждой метрики.
4. Открой доступные представления блока.
5. Наведи указатель на значок `i`, если он есть: Portal показывает полное название или методологическую подсказку.
6. Сопоставляй значение с периодом сравнения, а не только с цветом стрелки.

!!! warning "Не заменяет бухгалтерский отчёт"
    Дашборды предназначены для анализа. Используй финансовые значения для учётных решений только по утверждённому регламенту и вместе с соответствующим отчётом.
