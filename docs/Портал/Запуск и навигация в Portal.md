# Запуск и навигация в Portal

Инструкция помогает открыть Portal и перейти к нужной рабочей функции через верхнее меню или главную страницу.

## Перед началом

- Нужна действующая учётная запись Portal.
- Видимость разделов и доступность действий зависят от роли.

## Порядок действий

1. Открой `https://portal.mooon.by`.
2. Войди под своей учётной записью.
3. На главной странице выбери карточку нужной функции или открой её через верхнее меню.
4. После завершения работы используй `выйти`.

![Главная страница Portal](../assets/media/portal/portal_home_ru.png)

Логотип `mooon` возвращает на главную страницу. Карточки главной открывают те же функции, что и выпадающие пункты верхнего меню. `Выйти` завершает сеанс Portal.

## Верхнее меню

| Пункт | Что открывает |
|---|---|
| `аналитика` | `Кинопульс`, `Ресторан`, `Маркетинг` |
| `управление` | бонусы, новости, цифровая реклама, роли, пользователи, возврат билетов, активация сертификатов, киоск |
| `поиск` | поиск билетов, сертификатов и промокодов |
| `каталог` | продукты и связанные справочники |
| `очистить чек` | очистка чека по кассовой зоне |
| `отчеты` | MOOON, Widget, ресторан, Ticket Reader |

## Если раздел не виден

Не открывай скрытый раздел по прямому адресу. Сначала проверь, что выбран правильный пункт меню, затем обратись к владельцу доступов для проверки роли.

Страницы `Цифровая реклама` и `Пользователи` сейчас находятся в состоянии разработки.

## Проверка результата

- Открылся экран выбранной функции.
- В верхнем меню подчёркнут активный раздел.
