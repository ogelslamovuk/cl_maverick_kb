# Демо: работа виджета на go2.by

## Навигация

Заходим на главную страницу **go2.by**

### Главная страница

На главной странице отображаются:
- **баннеры** с анонсами фильмов
- **календарь**, который можно пролистывать стрелочками
- **события** — плитки с фильмами доступными на выбранную дату

### Выбор билета

Для выбора билета можно:
- нажать на баннере кнопку **«Купить билет»**
- нажать на **само событие**

→ Открывается **страница события**

### Страница события

На странице события:
- **трейлер** фильма
- **актуальные даты**, когда есть сеансы
- **объекты** — указаны залы
- **время сеансов**
- ниже — **описание** самого фильма

Выбираем → объект → зал → время сеанса

### Выбор мест

→ Открывается **раскладка зала**

- Выбираем места — они **подсвечиваются тёмным цветом**
- Переходим к оплате

### Оплата

Вводим **гостевой email** (и другие данные) → переход на платёжную страницу.
